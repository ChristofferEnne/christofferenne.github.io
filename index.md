    <p style="margin-top: 0;">Developed in Rust and inspired by MarkovJunior, Patchwork is a highly optimized procedural generation toolkit designed for building complex, logic-based game environments and data structures. Originally conceived as a dedicated Markov operations library, the project has evolved into a versatile, high-performance ecosystem featuring its own custom language parser. The suite is divided into two core components: a dedicated visual editor for authoring and debugging generation scripts, and a soon-to-be open-source runtime library built for frictionless integration into modern game engines.</p>
